Introduction¶

Top

Written Japanese uses a mixed-script system: kanjimorphographic Chinese characters, which is to say: graphemes that mainly represent the morphemes used in word formationand derived from these, there are two types of phonographic kana, which are simpler graphemes representing only the Japanese sound systems moraic syllable (e.g., ka) foundation, used mostly for grammatical functions.

Figure 1: A text containing archaic handwritten Japanese, via

Handwritten Japanese before the 19c standardizations of the Meiji era is characterized by myriad, idiosyncratic variations of kanji and kana (hentaigana), collectively called kuzushiji, and has thereby limited primary access to pre-modern corpora to the few who have attained the necessary literacy (Hashimoto, et al., 2017).

Given the value of the historical record, deciphering kuzushiji is a significant enterprise, enabled by scanning images of historical texts, with scholars painstakingly hand-labeling characters. A typical practice is training computational models to classify characters, trial-and-error supervised with expert labels acting as the ground truth, for corrective feedback. Examples of applications for this topic range from adapting Edo period recipes to gathering seismological data to better predict earthquakes.

Deep neural networks are state-of-the-art in image recognition, with multiple layers of interconnected neurons leveraged to learn numeric representations of training data (e.g., a subset of character images extracted from a corpus, and their manually coded target labels), which can be generalized to evaluate new data (e.g., classifying a novel subset of images and targets).

For image tasks, the standard architecture is the convolutional neural network (CNN). The key feature of its effectiveness is that a CNN retains the spatial relationships of the pixels comprising images, rather than flattening inputs at the onset.


Figure 2: A typical convolutional neural network, via

TensorFlow is a high-level library known for providing prefab functions executed over the abstraction of a computational graph. It is an accessible resource which allows diverse researchers to employ deep learning without attending to low-level details (e.g., convolutional operations).

Motivation¶

Top

Nominally, the aim of this projectoriginally for DePaul University's CSC 578, Fall 2017, and very much a work in progressis kuzushiji recognition with a convolutional neural network: the approach taken was to work within TensorFlow, but implementing the network from scratch, rather than using pre-built operations: to explicitly code such low-level components as the multiplication of tensor data structures by functions for convolution and subsampling, and arranging the data and control flow over bespoke forward and backward layering.

The primary purpose was self-directed learning. Building the network from scratch in TensorFlow, as opposed to NumPy or Octave, provided unique opportunities: to gain insight into the computational graph as a framework and to practice manipulating high-dimensional tensors, with the TensorFlow ecology acting as a kind of scaffolding for deconstructing pre-existing code.

The focus on classifying calligraphy enabled working with meaningful data in a generative exercise, motivating intensive practice with fundamentals. This includes the nuanced challenges and rewards of preprocessing multicultural artifacts.

Due to the workload and bugs introduced with this scratch-built, ad hoc approach and a good measure of domain ignorance, this implementation adheres to mostly vanilla structures in order to ease fine-tuning of the hyperparameters and focus on learning important aspects of CNNs.

Methods¶

Top

Following a standard template, the architecture consists of two convolutional and subsampling layer pairs, followed by a pair of affine layers. Non-linearities for the convolutional layers use an explicitly defined function for leaky rectified linear units (LReLU): the leaky version of this transfer function (intended to prevent the problem of exponentially decreasing gradients) mitigates the issue of cell death in standard ReLU by allowing for a minimal gradient in the negative case, rather than thresholding at zero. The convolutions are downsampled with conventional max-pooling.


Figure 3: Sample TensorBoard visualization

Dropout (Srivastava, et al., 2014) is applied to the first affine layer, also from scratch: dropout reduces neuronal co-adaptations by forcing cells to adapt to the more diverse ecologies created by randomly mutated subsets of a network, creating a robust combined model with more independent feature units.

Finally, the readout layer outputs are sent directly to an explicit softmax function with negative log-likelihood loss, to obtain and evaluate a probability distribution for the appropriate class labels. This function is defined such that it scales the logits before normalization, to avoid underflow and overflow, per a best practice referenced by Goodfellow, et al. (2016) (+ja).

Weight decay via L2 regularization is used to help the model better discover underlying patterns in the data with less sensitivity to local noise. Momentum is used with gradient descent, titrating the velocity through friction rather than merely force (the gradient). Of final note, the training phase employs early-stopping based on the validation set accuracy, aborting when this saturates.

def get_data(flist, label_dict):
    """Retrieve images and labels.
    
       Use file list to obtain resized grayscale images,
       and corresponding image classes via label dictionary.
       This is a modified function from http://codh.rois.ac.jp.
        
       Args:
           flist: Array of image path strings.
           label_dict: Dictionary:
                       {'char code': (class #, 'char')}
       Returns:
           Numpy arrays x, y containing:
               x: Images [pixel values--intensities here for grayscale], 
               y: Class numbers (integers).
    """
    x = np.zeros((len(flist), ih, iw))
    y = np.zeros((len(flist), 1))
    for idx, file in enumerate(flist):
        img = cv2.imread(file, 0) # as grayscale
        x[idx] = cv2.resize(img, dsize=(ih, iw))
        y_k = os.path.basename(file).split('_')[0] # get char code
        y[idx] = label_dict[y_k][0] # first element of tuple value is char #
    y = np.array(y, dtype='uint8').reshape((len(y), 1))
    return x, y

Forward Pass¶

Top

Using a memory-cheap size for mini-batches, we'll use tweaked functions provided from TensorFlow's MNIST tutorial to initialize parameters in a way that mitigates saturation.

I use the rectifier weight initialization to calibrate variances that is described as superior to, say, Xavier, in this paper (He, et al., 2015) and noted in the CS231N notes.

I am also using a small positive constant on the bias to ensure neurons initially fire for ReLU gradient computation, thereby avoiding "dead" units with no gradients: a best practice suggested in the Deep Learning book by Goodfellow, Bengio, and Courville (2016) and implemented in the TensorFlow MNIST tutorial.

def weight_variable(shape):
    with tf.name_scope('weights'):
        initial = tf.truncated_normal(shape, 
                                      stddev = np.sqrt(2.0 / shape[0]), 
                                      name='W')
    return tf.Variable(initial)

def bias_variable(shape):
    with tf.name_scope('biases'):    
        initial = tf.constant(0.1, shape=shape, 
                              name='b')
    return tf.Variable(initial)

def velocity_variable(shape):
    with tf.name_scope('velocities'):        
        initial = tf.zeros(shape, name='v')
    return tf.Variable(initial)

def variable_summaries(var):
    # Via TensorFlow tutorial:
    # https://www.tensorflow.org/tutorials/mnist/pros/#deep-mnist-for-experts    
    with tf.name_scope('summaries'):
        mean = tf.reduce_mean(var)
        tf.summary.scalar('mean', mean)
    with tf.name_scope('stddev'):
        stddev = tf.sqrt(tf.reduce_mean(tf.square(var - mean)))
    tf.summary.scalar('stddev', stddev)
    tf.summary.scalar('max', tf.reduce_max(var))
    tf.summary.scalar('min', tf.reduce_min(var))
    tf.summary.histogram('histogram', var)    
    
def get_dims(h, h2, w, w2, s, p):
    # (W - F + 2P/S + 1)
    out_h = (h - h2 + 2*p) // s + 1
    out_w = (w - w2 + 2*p) // s + 1    
    return out_h, out_w    
    
def im2col(X, h, w, d, out_h, out_w, s=1, p=1):
    # TensorFlow conversion of CS231N Numpy functions in im2col.py: 
    # http://vision.stanford.edu/teaching/cs231n/assignments.html
    # http://cs231n.github.io/assignments2017/assignment2/
    # http://cs231n.stanford.edu/assignments/2017/spring1617_assignment2.zip    
    X_p = tf.pad(X, [[0, 0], [p, p], [p, p], [0, 0]])
    # Slice offsets: s*oh:s*oh+kh, s*ow:s*ow+kw
    # Accounting for stride, moving from along output dims, excise
    # kernel-sized tiles from the input.    
    X_l = [tf.slice(X_p, [0, s*oh, s*ow, 0], [-1, h, w, -1])
           for oh in range(out_h) for ow in range(out_w)]
    # In a list of length: flattened output dims (# locations w/ s=1, p=0), 
    # we now have a tensor of kernel dims w/ 
    # no. samples, e.g., (100, 3, 3, 1)
    # Make into tensor of dims: kernel height, kernel width, kernel depth, 
    # (no. samples--e.g., batch_size), (output dims--e.g., 28*28=784)
    # e.g., 3  3  1  100  784
    X_mt = tf.transpose(tf.stack(X_l, axis=1), [2, 3, 4, 0, 1])
    # Shape into matrix:
    # Each column vector magnitude = flattened kernel dims (e.g., 3*3*1)
    # Each row vector magnitude = # of locations for the images (e.g., 100*784)
    # So each coordinate = image reflection in a kernel dimension   
    X_col = tf.reshape(X_mt, [d*h*w, -1])
    # Now we have a, for example, (3*3*1)  (100*28x*28) tensor to be transformed
    # k_dims_flat  num_locations
    return X_col    
    
def convolve_forward(X, W, b, stride=1, padding=1):
    # convolve == matrix multiply
    n, h, w, _ = X.get_shape().as_list()    
    kh, kw, kd, km = W.get_shape().as_list()
    # (W - F + 2P/S + 1)
    out_h, out_w = get_dims(h, kh, w, kw, 
                            stride, padding)
    X_col = im2col(X, kh, kw, kd, 
                   out_h, out_w, 
                   stride, padding)
    # transpose, shape weights into matrix (e.g., 32  3*3*1)
    W_row = tf.reshape(tf.transpose(W, [3, 0, 1, 2]), 
                       [km, kh*kw*kd])
    # (32  9) * (9  78400)
    z = tf.matmul(W_row, X_col) + tf.reshape(b, [-1, 1])
    # reshape to proper dimensions--e.g., 32  100  28  28
    z = tf.reshape(z, [km, n, out_h, out_w] )
    # transpose to n, out_h, out_w, km
    # 100  28  28  32
    z = tf.transpose(z, [1, 2, 3, 0])
    cache = (X, X_col, W, W_row, stride, padding)
    return z, cache

def leaky_relu(X, eps=0.2):
    return tf.maximum(X, eps * X), (X, eps)

def max_pool_forward(X, pool_h=2, pool_w=2, stride=2, padding=0):
    n, h, w, km = X.get_shape().as_list()
    # 100  28  28  32
    out_h, out_w = get_dims(h, pool_h, w, pool_w, stride, padding)
    X_col = im2col(X, pool_h, pool_w, km, 
                     out_h, out_w, stride, padding)
    # X_col example shape: (2*2*32)  (100*14*14)
    prop_d = [pool_h * pool_w, km, n, out_h, out_w]
    X_col = tf.reshape(X_col, prop_d)
    # X_col now would be [4  32  100  14  14]
    pool = tf.reduce_max(X_col, axis=0)
    # pool: [32  100  14  14]
    # transpose pool to: [100  14  14  32] or: [n, h, w, m]
    pool = tf.transpose(pool, [1, 2, 3, 0])
    return (pool, 
            (X, X_col, pool, pool_h, pool_w), n)

def affine_forward(X, W, b, n):
    X_fc = tf.reshape(X, [n, -1])
    z = tf.matmul(X_fc, W) + b
    return z, (X_fc, W)

def dropout_forward(X, keep_prob=0.5):
    # Attempting to implement inverted dropout
    # The binary mask selects a percentage of values to be masked to 0
    # (e.g., keep_prob > 50% means random floats added to it 
    # can be smaller and still likely end up > 1,
    # rather than being 50% likely or more to be closer to 0); 
    # if keep_prob isn't 0 (avoid zero division),
    # we scale up the remaining training phase activations by keep_prob 
    # so that the val/test phase activations without
    # dropout don't have to be scaled down to account for 
    # their undropped neuron outputs.
    mask = tf.floor(tf.random_uniform(tf.shape(X), dtype=X.dtype) + keep_prob)
    mask *= (1. / keep_prob if keep_prob is not None else 0.)
    return X * mask, mask

def softmax_cross_entropy(logits, y_):
    # numerically stable
    logits_scaled_exp = tf.exp(logits - tf.reduce_max(logits))
    y = tf.transpose(tf.transpose(logits_scaled_exp) 
                            / tf.reduce_sum(logits_scaled_exp, axis=1))
    with tf.name_scope('total'):
            cross_entropy = -tf.reduce_mean(tf.reduce_sum(y_ * tf.log(y), 
                                                                 axis=1))
    return y, cross_entropy
    
def forward_pass(X, layers, parameters, velocities, s=1, p=1):
    caches = dict()
    for layer in layers:
        with tf.name_scope(layer + '_forward'):
            if layer == 'conv_relu1':
                with tf.variable_scope('conv_relu1') as scope:
                    W1 = weight_variable([kf, kf, c, fm])
                    vW1 = velocity_variable([kf, kf, c, fm])    
                    b1 = bias_variable([fm]) 
                    vb1 = velocity_variable([fm])
                
                variable_summaries(W1)
                variable_summaries(b1)
                
                parameters.append(W1)
                parameters.append(b1)
                velocities.append(vW1)
                velocities.append(vb1)
                
                with tf.name_scope('conv1_preactivations'):
                    conv1_layer, conv1_cache = convolve_forward(X, W1, b1, 
                                                              stride=s, 
                                                              padding=p)
                    tf.summary.histogram('conv1_preactivations', conv1_layer)
                with tf.name_scope('conv1_activations'):
                    conv1_activation_layer, conv1_relu_cache = leaky_relu(conv1_layer)
                    tf.summary.histogram('conv1_activations', 
                                         conv1_activation_layer)
                
                caches['conv_relu1'] = (conv1_cache, conv1_relu_cache)
                
            if layer == 'pool1':
                pool1, pool1_cache, n = max_pool_forward(conv1_activation_layer)
                _, pool1_h, pool1_w, pm1 = pool1.get_shape().as_list()
                
                caches['pool1'] = pool1_cache
                
            if layer == 'conv_relu2':
                with tf.variable_scope('conv_relu2') as scope:
                    W2 = weight_variable([kf, kf, pm1, fm2])
                    vW2 = velocity_variable([kf, kf, pm1, fm2])
                    b2 = bias_variable([fm2])
                    vb2 = velocity_variable([fm2])  

                variable_summaries(W2)
                variable_summaries(b2)

                parameters.append(W2)
                parameters.append(b2)
                velocities.append(vW2)
                velocities.append(vb2)
                
                with tf.name_scope('conv2_preactivations'):
                    conv2_layer, conv2_cache = convolve_forward(pool1, W2, b2, 
                                                              stride=s, 
                                                              padding=p)
                    tf.summary.histogram('conv2_preactivations', conv2_layer)
                with tf.name_scope('conv2_activations'):
                    conv2_activation_layer, conv2_relu_cache = leaky_relu(conv2_layer)
                    tf.summary.histogram('conv2_activations', 
                                         conv2_activation_layer)
                
                caches['conv_relu2'] = (conv2_cache, conv2_relu_cache)
                
            if layer == 'pool2':
                pool2, pool2_cache, n = max_pool_forward(conv2_activation_layer)
                _, pool2_h, pool2_w, pm2 = pool2.get_shape().as_list()
                
                hidden_nodes = 1024
                flat = [pool2_h*pool2_w*pm2, hidden_nodes]
                with tf.variable_scope('pool2') as scope:                
                    W3 = weight_variable(flat)
                    vW3 = velocity_variable(flat)
                    b3 = bias_variable([hidden_nodes])
                    vb3 = velocity_variable([hidden_nodes])  

                variable_summaries(W3)
                variable_summaries(b3)

                parameters.append(W3)
                parameters.append(b3)
                velocities.append(vW3)
                velocities.append(vb3)   
                
                caches['pool2'] = pool2_cache                  
                
            if layer == 'fc1_relu_dropout':
                with tf.name_scope('fc_preactivations'):
                    fc1, fc1_cache = affine_forward(pool2, W3, b3, n)                
                    tf.summary.histogram('fc_pre_activations', fc1)
                with tf.name_scope('fc_activations'):                    
                    fc1_activation, fc1_relu_cache = leaky_relu(fc1)  
                    tf.summary.histogram('fc_activations', fc1_activation)    
                with tf.name_scope('fc_dropout'):                
                    fc1_dropped, fc1_dropout_cache = dropout_forward(fc1_activation, 
                                                                keep_prob) 
                    tf.summary.histogram('fc_dropout', fc1_dropped) 
                output_nodes = len(labels) 
                
                with tf.variable_scope('fc1_relu_dropout') as scope:                
                    W4 = weight_variable([hidden_nodes, output_nodes])
                    vW4 = velocity_variable([hidden_nodes, 
                                        output_nodes])    
                    b4 = bias_variable([output_nodes])
                    vb4 = velocity_variable([output_nodes])    
                
                variable_summaries(W4)
                variable_summaries(b4)  
                
                parameters.append(W4)
                parameters.append(b4)
                velocities.append(vW4)
                velocities.append(vb4)   
            
                caches['fc1_relu_dropout'] = (fc1_cache, fc1_relu_cache, 
                                              fc1_dropout_cache)
                
            if layer == 'fc2_readout':
                
                with tf.name_scope('fc2'):
                    fc2, fc2_cache = affine_forward(fc1_dropped, W4, b4, n)                
                caches['fc2_readout'] = fc2_cache
                tf.summary.histogram('fc2', fc2)   
                    
                with tf.name_scope('cross_entropy'):                    
                    y, loss = softmax_cross_entropy(fc2, y_)
                    tf.summary.scalar('cross_entropy', loss)             
            
    return parameters, velocities, y, loss, caches

def col2im(delta, kh, kw, kd, n, oh, ow, stride, padding):
    oh = oh + (2 * padding)
    ow = ow + (2 * padding)
    X_im = im2col(delta, kh, kw, kd, oh, ow, stride, 2 * padding)
    X_im = tf.reshape(X_im, [kh*kw*kd, -1, oh, ow])
    X_im = tf.transpose(X_im, [1, 2, 3, 0])    
    if padding > 0:
        X_im = X_im[:, padding:oh-padding, padding:ow-padding, :] 
    return X_im
    
def affine_backward(delta, cache):
    X, W = cache
    h_delta_tmp = tf.matmul(delta, tf.transpose(W))
    ndW = tf.matmul(tf.transpose(X), delta)
    ndb = tf.reduce_sum(delta, axis=0)
    return h_delta_tmp, ndW, ndb

def leaky_relu_prime(cache):
    # Values are  if x is negative, 1 otherwise.
    X, eps = cache
    dX = tf.where(X < 0, 
                  tf.constant(eps, shape=X.shape), 
                  tf.ones(shape=X.shape))
    return dX

def pool_backward(delta, cache):
    X, X_col, pool, pool_h, pool_w = cache
    n, h, w, c = X.get_shape().as_list()
    pn, ph, pw, pm = pool.get_shape().as_list()
    delta_r = tf.reshape(delta, [n, ph, pw, pm])
    dX_col = tf.zeros_like(X_col)
    dX_col = tf.transpose(dX_col, [0, 2, 3, 4, 1]) 
    X_col2 = tf.transpose(X_col, [0, 2, 3, 4, 1])
    pool_filter = X_col2 + pool
    pool_filter = pool_filter / 2
    pool_cond = tf.equal(tf.cast(X_col2, dtype=pool_filter.dtype), 
                         pool_filter)
    dX_im = tf.where(pool_cond, tf.ones_like(X_col2), dX_col)
    dX_im = delta_r * dX_im
    dX_im = tf.transpose(dX_im, [1, 2, 3, 4, 0])
    dX_im = tf.reshape(dX_im, [n, pool_h*ph, pool_w*pw, c])
    # dX_im should now be expanded tensor of zeros,
    # blocking the error signal, except where
    # error indices match max pool indices;
    # so delta has been expanded to proper dims. 
    return dX_im

def conv_backward(delta, cache):
    X, X_col, W, W_row, stride, padding = cache
    n, h, w, c = X.get_shape().as_list()
    kh, kw, kd, km = W.get_shape().as_list()
    dX_im = col2im(delta, kh, kw, kd, n, h, w, stride, padding)
    dX_im = tf.reshape(dX_im, [km, -1])
    dX_im = tf.matmul(tf.transpose(dX_im), W_row)
    dX_im  = tf.reshape(dX_im, [n, h, w, -1, c])
    dX_im = tf.reduce_max(dX_im, axis=3)
    dX_col = tf.reshape(delta, [-1, km])
    ndW = tf.matmul(X_col, dX_col)
    ndW = tf.reshape(ndW, [kh, kw, kd, km])
    ndb = tf.reduce_sum(delta, axis=(0, 1, 2))
    return dX_im, ndW, ndb

def backward_pass(y, layers, caches):
    gradients = []
    with tf.name_scope('out_delta'):             
        error = (y - y_) / batch_size        
        tf.summary.histogram('out_delta', error)     

    for layer in layers[::-1]:
        with tf.name_scope(layer + '_back'):
            if layer == 'fc2_readout':
                with tf.name_scope('fc2_delta'):
                    h_delta_tmp, ndW4, ndb4 = affine_backward(error, 
                                                              (caches['fc1_relu_dropout'][1][0], 
                                                               caches['fc2_readout'][1]))
                    h_delta4 = tf.multiply(h_delta_tmp, 
                                          leaky_relu_prime(caches['fc1_relu_dropout'][1]))    
                    # Dropout adjustment
                    h_delta4 *= caches['fc1_relu_dropout'][2]
                    tf.summary.histogram('fc2_delta', h_delta4)

                    variable_summaries(ndb4)
                    variable_summaries(ndW4)

                    gradients.append(ndb4)
                    gradients.append(ndW4)     

            if layer == 'fc1_relu_dropout':

                    h_delta3, ndW3, ndb3 = affine_backward(h_delta4, caches['fc1_relu_dropout'][0])

                    variable_summaries(ndb3)
                    variable_summaries(ndW3)

                    gradients.append(ndb3)
                    gradients.append(ndW3)  
                    
            if layer == 'pool2':

                    h_delta_tmp = pool_backward(h_delta3, caches['pool2'])
                    h_delta2 = tf.multiply(h_delta_tmp, 
                                          leaky_relu_prime(caches['conv_relu2'][1]))

                    variable_summaries(ndW2)

                    gradients.append(ndb2)
                    gradients.append(ndW2)

                    h_delta1 = tf.multiply(h_delta_tmp,

                    variable_summaries(ndb1)
                    variable_summaries(ndW1)

                    gradients.append(ndb1)
                    gradients.append(ndW1)

# Stochastic gradient descent over mini-batches w/ vanilla momentum, L2
def sgd_with_momentum(velocities, parameters, gradients, eta=0.001, mu=0.9):
    return [parameter.assign((1 - eta * (lmbda / X_train.shape[0])) *
                             parameter + v.assign(mu * v - 
                                                  ((eta / batch_size) * gradient))) 
            for parameter, gradient, v in zip(parameters, 
                                              gradients[::-1], 
                                              velocities)]
      
with tf.name_scope('train'):
    trainer = tf.group(*sgd_with_momentum(velocities, parameters, gradients, 
                                          eta=0.01, mu=0.5))   
    
with tf.name_scope('accuracy'):
    pred = tf.argmax(y, axis=1)
    act = tf.argmax(y_, axis=1)
    with tf.name_scope('correct_prediction'):
        correct_prediction = tf.cast(tf.equal(pred, act), tf.float32)
    with tf.name_scope('accuracy'):
        accuracy = tf.reduce_mean(correct_prediction)    
    tf.summary.scalar('accuracy', accuracy)

def get_n_batches(X):
    return int(X.shape[0] // batch_size)

def get_mini_batches(X, y_):
    num_batches = get_n_batches(X)
    for i in range(0, num_batches * batch_size, batch_size):
        if i == (batch_size * (num_batches - 1)):
            # last batch, fill w/ samples left
            if len(X[i:]) <= batch_size:
                xt, yt = X[i:], y_[i:]
        else:
            xt, yt = X[i:batch_size+i], y_[i:batch_size+i]
    yield xt, yt

As noted, this project was an educational exercise first, and a kind of pilot study second, using small sets for runs, to obtain rapid results for debugging (a work in progress). A simple summary would be that the scratch-built network appears to work well with small sets of the simpler, frequent kana (more specifically, hentaigana), and would require fine-tuning, redesign, and refactoring to broaden its expressive capacity and increase robustness.

Discussion¶

Top

Figure 7: A sample characters activations for the first convolutional layer w/ 32 feature maps

There is a wealth of research on the benefits of learning through productive, generative, meaningful tasks, and in that sense, this project was of great value. Learners tend to think they understand more than they do: illusions of mastery [PDF] that encourage them to passively engage with materials, and commonly it isnt until they practice in a generative, productive rather than receptive way that they notice the gaps in their comprehension. Doing this in TensorFlow, rather than Octave or NumPy, for example, presented a fine opportunity to connect various types of comprehension (e.g., intuitive descriptions, mathematical formulae, and code at multiple levels of abstraction), a kind of elaborative learning w/ convergent cognition.

On the other hand, it is quite impractical to re-invent the wheel, and in terms of the industry, familiarity with the standard use of TensorFlow and other libraries is ideal. At the same time, a happy medium can be found, pedagogically, with a progressive, piecewise design of such an activity; and in terms of practical use, it is worthwhile to cultivate a bottom-up perspective, an anodyne to the adverse effects of blackboxingespecially when integrating deep learning into sundry disciplines with varying conceptual models, problems, and data. The potential for creativity in the design of neural networks in the context of the Digital Humanities, is quite vast.

In the future, I would like to experiment in a similar fashion with the modern ETL Character Database, using Geoffrey Hintons capsule architecturewhich I had encountered through one of his recorded talks prior to CNNs when I first began my studies in this areanow that new papers have been published to detail the work of researchers such as Sara Sabour, and various explainers and code repositories have been released.

My own mental models of reading Japanese stem from research I had previously read from authors such as Koji Miwa and Harald Baayen (known for his grammatical work, as well). To fully explore the applications of such biological models to network design, integrating attention and some notion of priority in layer-to-layer activations might be necessary, such that one might quantify the character-driven processing and submorphemic supplementation thereof.

There have been putatively physiological elastic distortions used to artificially expand MNIST data; I have considered a more explicit modeling of such a process, as it seems a promising avenue, given the sensorimotor complexities of kanji handwriting; recent RNN research on calligraphic stylisation could be helpful for this.

For this project there are numerous improvements to be made. Gradient checking would be useful, and as the prefab Adam Optimizer with its parameter-independent, adaptive learning rate worked well when used as a comparison to the scratch-built optimizer, I believe it would be worthwhile to implement it from scratch. Batch normalization might also be useful.

Running the network on full datasets via AWS/GPU required changes to the code that need to be integrated into the code presented here. Other changes include a proper implementation of the im2col and col2im operations.

In general, this work is part of a field of recognized importance: for instance, Dr. Regina Barzilay recently won a MacArthur genius grant.
